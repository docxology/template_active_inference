# Section fragments (multi-track sources)

Per-section, per-track source fragments composed into the manuscript by the sheaf
composer. Each section directory holds one fragment per participating track
(e.g. `prose.md`, `formalism.md`, `gnn.md`, `ontology.yaml`, `simulation.md`,
`visualization.md`, `pymdp.md`, `lean.md`).

- `preliminaries/` — setup and notation.
- `bernoulli_coupling/` — the Bernoulli coupling model section.
- `sophisticated_inference/` — the pymdp sophisticated-inference T-maze section.
